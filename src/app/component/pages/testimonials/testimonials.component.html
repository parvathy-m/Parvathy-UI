Levica

This system aims to streamline transactions between registered business users and customers. Users may apply for different contracts and asset transfers (secondary transactions) by utilizing security tokens that are linked with overseas financial systems. The system enables users to execute payments via electronic currency using their smartphones. Transactions can be settled through bank transfers or ETH coins. After completion, users receive Levica points, which they can redeem for cash.

Quick CROSS Web

Quick CROSS Web is a comprehensive application intended to edit, compile, and organize questionnaire data, primarily collected through Macromill. The data is then assembled into a report that end-users can utilize for various purposes, such as presentations or conferences. Additionally, the system functions as an in-house application to generate aggregates requested by the end users.
